About

The HappyBrackets project makes creative coding on multiple devices like Raspberry Pi and Beagle Bone easy. With HappyBrackets, you can set them all up once and then remote control them from a single computer. HappyBrackets is an open source project. Developers wishing to hack or get involved can find everything they need on our GitHub page.
